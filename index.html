Prompt Pack + Custom Agent

Steno Mode makes technical AI answers shorter without making them worse.

Steno Mode, also called Stenographer Mode, is a prompt pack and custom agent for readable token reduction in technical workflows. It is built for VS Code Copilot, ChatGPT custom instructions, Claude system prompts, and Cursor rules.

FAQ

Questions people actually search

What is the difference between Caveman and Steno Mode?

Caveman cuts grammar aggressively. Steno Mode compresses structure while preserving more readability and technical precision.

Can I use this with GitHub Copilot and ChatGPT?

Yes. The repo ships prompt and pack surfaces for GitHub Copilot, ChatGPT, Claude, and Cursor.

Is this a model or a prompt?

It is a prompt-driven response style, not a separate model. The repo provides prompt packs, agents, and install flows.
